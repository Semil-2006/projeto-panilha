Produto Processor

Sistema Avançado de Análise e Processamento de Dados de Produtos e Serviços

Visão Geral

O ProdutoProcessor é uma classe Python projetada para processar, analisar e gerar relatórios sobre dados de produtos e serviços. Ele oferece funcionalidades completas desde o carregamento de dados até a geração de relatórios profissionais e visualizações gráficas.

Carregamento de Dados

Valida e carrega dados de produtos a partir de arquivos Excel, com verificação de integridade e estrutura.

Cálculos Automáticos

Realiza cálculos complexos como aplicação de impostos, margens de lucro e classificação por categorias.

Análise Estatística

Gera estatísticas detalhadas sobre os produtos, incluindo médias, totais e distribuição por categorias.

Relatórios Profissionais

Exporta dados para Excel com formatação profissional, incluindo múltiplas abas e estilos.

Visualização de Dados

Cria gráficos e visualizações para melhor compreensão dos dados processados.

Logging Completo

Registra todas as operações em arquivo de log e console para rastreamento e depuração.

Como Usar

Exemplo Básico de Uso

# Importar a classe
from produto_processor import ProdutoProcessor

# Criar instância do processador
processor = ProdutoProcessor("Produtos.xlsx")

# Carregar e processar dados
if processor.carregar_dados() and processor.processar_dados():
    # Gerar relatório e gráficos
    processor.gerar_relatorio("Relatorio_Produtos.xlsx")
    processor.gerar_graficos("graficos/")

Nota: O arquivo de entrada deve conter pelo menos as colunas: Produtos, Preço Base Original, Tipo e Multiplicador Imposto.

Documentação da Classe

Métodos Principais

__init__(arquivo_entrada: str)
Inicializa o processador com o arquivo de entrada.
arquivo_entrada: Caminho para o arquivo Excel contendo os dados de produtos.
carregar_dados() -> bool
Carrega e valida os dados de entrada. Retorna True se bem-sucedido.
Retorna: True se os dados foram carregados com sucesso, False caso contrário.

Validação: Verifica se o arquivo existe, se contém as colunas obrigatórias e se os valores numéricos são válidos.

processar_dados() -> bool
Processa os dados aplicando regras de negócio e cálculos. Retorna True se bem-sucedido.
Retorna: True se o processamento foi concluído com sucesso, False caso contrário.

Este método realiza várias operações incluindo aplicação de impostos, cálculo de margem de lucro e classificação por categorias de preço.

gerar_relatorio(arquivo_saida: str) -> bool
Gera relatório completo em Excel com formatação profissional.
arquivo_saida: Caminho para o arquivo Excel de saída.
Retorna: True se o relatório foi gerado com sucesso, False caso contrário.

O relatório inclui múltiplas abas com diferentes análises e é formatado automaticamente.

gerar_graficos(diretorio_saida: str = '.') -> Dict[str, str]
Gera gráficos de análise e retorna caminhos dos arquivos.
diretorio_saida: Diretório onde os gráficos serão salvos (padrão: diretório atual).
Retorna: Dicionário com os caminhos dos arquivos de gráficos gerados.

Métodos Internos

_aplicar_impostos() -> None
Aplica regras de impostos diferenciadas para produtos (12%) e serviços (50%).

Tipos inválidos são registrados em log mas não interrompem o processamento.

_classificar_categorias_preco() -> None
Classifica produtos em categorias de preço: Baixo (0-100), Médio (100-500), Alto (500+).
_calcular_estatisticas() -> None
Calcula estatísticas avançadas sobre os dados processados.

As estatísticas incluem totais, médias, valores extremos e análises por categoria.

_formatar_excel(arquivo: str) -> None
Aplica formatação profissional ao arquivo Excel gerado.

Inclui formatação de cabeçalhos, ajuste de colunas e formatação numérica.
